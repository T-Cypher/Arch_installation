Arch_installation
=================

Personal installation of ArchLinux distro

Usage
=====

With git
    
    # upgrade system
    pacman -Syu
    # install git
    pacman -S git
    # get the script
    git clone git://github.com/T-Cypher/Arch_installation

Without git

    # upgrade system
    pacman -Syu
    # get the script
    wget --no-check-certificate https://github.com/T-Cypher/Arch_installation/archive/master.zip -O - | tar xz
